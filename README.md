# Cracktron
cracktro style effects for unity

## Dependencies
[Unity](http://www.unity3d.com) 2017.1.0f3

## Getting Started
Open the root folder of the project in Unity.
Pick a demo folder from one in Assets/Demos.
Open main.scene and play around!
